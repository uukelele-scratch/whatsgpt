## Features
* Chat history management
* Image vision & recognition
* Works when owner is offline
* Understanding who sent which messages 
* Viewing replies to messages
* Choosing not to respond to messages.
* Image Generation (though only uses link for now)
* Can listen to audio and voice messages!
* Can watch videos under 20MB!

```
$ git clone https://github.com/uukelele-scratch/whatsgpt.git
$ cd whatsgpt
$ cp .env.example .env
$ # Edit your env file.
```
